# AlyraSolanaProject

## Development

### Frontend

```shell
pnpm run dev
```

### Build frontend

```shell
pnpm run build
```

### Tests

To launch the tests, run:

```shell
# ./.
anchor test
```


## Deploy frontend to production

Just push to main or merge pull request into main, if code can build without errors then Vercel deploys it
